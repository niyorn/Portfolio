PIAN0.

BEKIJK SITE BEKIJK CODE

Pian0 is een digitale piano dat via keyboard of muis bespeeld kan worden.

Dit is het eerste project waar ik in aanraking met javascript kwam. Bij dit project worden dingen als variable en functies uitgelegd. Om te laten zien dat je de basis beheerst moest er een applicatie bedacht worden die interactief is.

De bedacht applicatie is een piano die gespeeld kan worden doormiddel van de keyboard of muis.
