NSVERHALEN

BEKIJK SITE BEKIJK CODE

NS verhalen is een site waar reizigers verhalen kunnen lezen die hun helpt om reistijd te doden.

Bij dit project moest er individueel een website ontworpen en gemaakt worden. De site dient daarnaast ook reponsive te zijn. De vormgeving van de site moest een uitstraling hebben die overeenkomt met een zelfgekozen organisatie.

Mijn rol binnen het project

  • Design
  • Programeren
  • Interaction

WERKWIJZE.=

Styleguide

De gekozen organisatie was NS. Als eerst is er onderzoek gedaan naar de huistijl van NS. Van die onderzoek is een styleguide gemaakt.

Micro interaction

Vervolgens worden er scenarios geschets waarbij gebruikers interacties hebben met de applicatie. Uit de scenarios bedenken wij een micro interactie die de gebruiker helpt om een interactie te voltooien.

Design/Coderen

Na het onderzoek was de taak om de site te ontwerpen. Bij het ontwerpen van de site is mobile first approach gebruikt. En natuurlijk is de laatste stap het maken van de site.

Wat heb ik geleerd?

De technieken waar ik voor het eerst in aanraken gekomen ben zijn: Git, Flexbox, Media query en de Mobile first approach. Deze technieken hebben mij erg geholpen met het repsonsive maken van de website.

Wat beter kon, was het designen en het maken van de site. Deze is niet genoeg apart gedaan, hierdoor moest er soms nog snel iets bedacht worden.
