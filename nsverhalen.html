NSVERHALEN

BEKIJK SITE BEKIJK CODE

NS verhalen is een site waar reizigers verhalen kunnen lezen die hun helpt om reistijd te doden.

Bij dit project moest er individueel een website ontworpen en gecodeerd worden. De site dient daarnaast ook reponsive te zijn. De vormgeving van de site moest een uitstraling hebben die overeenkomt met een zelfgekozen organisatie.

Mijn rol binnen het project

  • Design
  • Programeren
  • Interaction

WERKWIJZE

Styleguide

De gekozen organisatie was NS. Als eerste is er onderzoek gedaan naar de huistijl van NS. Van dat onderzoek is een styleguide gemaakt.

Micro interacties

Vervolgens heb ik situaties geschets waarbij gebruikers interacties hebben met de applicatie. Uit de scenarios worden er micro interacties bedacht die de gebruiker helpt met het voltooien van zijn of haar taak.

Design/Coderen

Na het onderzoek was de taak om de site te ontwerpen. Bij het ontwerpen van de site is mobile first approach gebruikt. Er is eerst gekeken hoe de site op een klein formaat uit moet zien en vanaf daar naar grotere formaten ontwerpen.

En de laatste stap is het coderen van de site. Bij het coderen van de site heb ik voor het eerst gebruikt kunnen maken van Flexbox. Flexbox heeft geholpen met het positioneren van elementen. Naast Flexbox heb ik ook kennis kunnen maken met media queries. De media queries is toegepast om de site responsive te krijgen.

Wat heb ik geleerd?

De technieken waarmee ik voor het eerst in aanraken ben gekomen zijn: Git, Flexbox, Media query en de Mobile first approach. Deze technieken hebben mij geholpen met het repsonsive maken van de website.
